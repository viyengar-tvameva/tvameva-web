## Rules Established This Session

### Deployment
- NEVER deploy without explicit user approval
- ALWAYS test before deploying
- Sequence: code ‚Üí typecheck ‚Üí build ‚Üí test ‚Üí user review ‚Üí user approves ‚Üí deploy

### Content
- NEVER fabricate testimonials, quotes, metrics
- Hero subheadlines = customer pain only
- Market stats = business value, not vendor stats
- "Customer Success" not "Results"
- "Executive Scorecards" not "Executive Dashboards"
- PropelEdge = "Revenue Orchestration" (12+ teams), not "Pre-Sales Automation"
- No company size restrictions ($500M-$1B)
- Only 3 visible SAs and 3 visible partners
- When user provides copy direction, elevate ‚Äî don't paste verbatim

### Design
- All cards in grids must be equal height
- No line-clamp on market context or proof point text
- Visual separation between sections (borders, backgrounds)
- Every SA page must have unique hero content (no cross-duplication)
- CTAs: Primary = Calendly demo, Secondary = SA assessment ‚Üí /contact
